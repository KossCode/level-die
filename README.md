# LEVEL LIE

Коварный мобильный платформер в духе *Level Devil*.  
Работает на iPhone в Safari и ставится на домашний экран как приложение (PWA).

## Как играть на iPhone

1. Открой `index.html` через любой локальный сервер (или залей репозиторий на хостинг).
2. В Safari: **Поделиться → На экран «Домой»**.
3. Управление: ◀ ▶ и **прыг**.

## Запуск локально

```bash
python3 -m http.server 8080
```

Открой http://localhost:8080 на телефоне (в той же сети) или в браузере.

## Что внутри

- 8 уровней с обманами: фейковые двери, падающие мосты, невидимый пол, разворот управления, убегающая дверь, фейковый экран победы
- Тач-управление под мобильные
- Офлайн через service worker

## Про нативный iOS

В этой среде нет Xcode — нельзя собрать `.ipa` / App Store билд.  
Игра сделана как **мобильный веб / PWA**, что на iPhone ощущается как приложение.

Если нужен нативный Swift/SpriteKit проект под Xcode — напиши, сделаем обёртку или порт.
